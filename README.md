# 영화 예약 프로그램
### 개발 환경
- Eclipse, Java(jdk8), oracle, mac

### 주요 기능
- 로그인 / 회원가입
- db에 저장된 영화 목록 리스팅
- 영화number에 대한 상영 일정 리스팅
- 일정number 및 좌석을 입력받아 영화 예약
